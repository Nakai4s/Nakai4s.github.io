Gitコマンド手順書

git init
役割：リポジトリを初期化する
使用例：git init
git clone
役割：リモートリポジトリをローカルにコピーする
使用例：git clone https://github.com/username/repository.git
git add
役割：変更をステージに追加する
使用例：git add .
git commit
役割：ステージに追加された変更をコミットする
使用例：git commit -m "コミットメッセージ"
git push
役割：ローカルリポジトリの変更をリモートリポジトリに反映させる
使用例：git push origin master
git pull
役割：リモートリポジトリから最新の変更をローカルに反映する
使用例：git pull origin master
git status
役割：リポジトリの状態を確認する
使用例：git status
git branch
役割：現在のブランチを確認、またはブランチを作成する
使用例：git branch
ブランチ作成: git branch 新しいブランチ名
git checkout
役割：ブランチを切り替える
使用例：git checkout 新しいブランチ名
git merge
役割：他のブランチの変更を現在のブランチに統合する
使用例：git merge 他のブランチ名
